# Profiler (профилировщик)

## Назначение

Автозагрузочный узел для замера времени выполнения участков кода.

## Использование

- `Profiler.start("имя")` — начать замер.
- `Profiler.end("имя")` — закончить замер.
- `enabled` — вкл/выкл (по умолчанию выключен; переключается по Shift+PageUp).
- `clear()` — сброс.

Системы вызывают `Profiler.start()`/`end()` напрямую. Profiler — autoload (всегда доступен), внутри `start()`/`end()` проверяется `if not enabled: return`, поэтому внешняя проверка не нужна.

**Важно:** `Engine.has_singleton("Profiler")` нельзя использовать — autoload-ы не являются Engine singletons и проверка всегда вернёт false.
