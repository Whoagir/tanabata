# Auriga (башня и система)

## Назначение

**Auriga** — атакующая башня (def_id: TOWER_AURIGA). Стреляет линией до 5 гексов в одном из шести направлений. Наносит урон всем врагам на линии за один «выстрел». Механика будет развиваться.

## Система

**AurigaSystem** (`core/systems/auriga_system.gd`) обновляется каждый кадр в фазе WAVE. Работает только для активных башен Auriga (подключённых к энергосети, с рудой).

## Поведение

- **Направление:** башня выбирает направление по количеству врагов на линии (максимум врагов). При равенстве — кратчайший поворот от текущего направления; при 180 градусах — случайный выбор.
- **Поворот:** один шаг поворота раз в несколько кадров (ROTATION_DELAY_FRAMES), то есть поворот замедлен.
- **Линия:** до LINE_LENGTH гексов от башни по выбранному направлению. Не проходит через занятые гексы (стены/башни) — линия обрывается.
- **Урон:** множитель базы (AURIGA_DAMAGE_MULT), общий нерф (AURIGA_NERF_MULT). Бонус к врагам типа «Тьма» (AURIGA_VS_DARKNESS_MULT). PURE-урон учитывает get_pure_damage_resistance; расход руды и проклятия — через GameManager (get_curse_extra_ore_per_shot и т.д.).

## Данные в ECS

- **auriga_lines** (словарь по tower_id): is_visible, hexes (массив гексов линии), current_direction, rotation_wait. Используется системой для плавного поворота и рендерером для отрисовки линии.

## Рендеринг

EntityRenderer отрисовывает линии Auriga: для каждой записи в ecs.auriga_lines с is_visible рисуется цепочка гексов (полигоны по hex.to_pixel). Слой — тот же, что и башни/враги (или отдельный подслой по z_index).

## Инициализация

GameRoot создаёт AurigaSystem (с power_finder для проверки питания) и вызывает auriga_system.update(scaled_delta) в главном цикле.
