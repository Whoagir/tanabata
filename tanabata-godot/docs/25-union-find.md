# Union-Find (система непересекающихся множеств)

## Назначение

Используется в EnergyNetworkSystem для объединения башен в связные компоненты при построении MST (минимальное остовное дерево).

## Операции

- **MakeSet** — создать множество для одного элемента.
- **Find** — найти представителя (корень) множества.
- **Union** — объединить два множества.

## Как используется

При добавлении рёбер в MST проверяется, не создадут ли они цикл. Если два конца ребра уже в одном множестве — цикл будет, ребро пропускается. Иначе — Union двух множеств, ребро добавляется.
