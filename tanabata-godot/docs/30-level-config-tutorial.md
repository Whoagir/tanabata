# LevelConfig и обучение (туториал)

## Назначение

**LevelConfig** (`core/level_config.gd`) задаёт параметры карты и пошаговые подсказки для основной игры и для уровней обучения (туториал). Данные волн, башен и врагов берутся из DataRepository (waves.json и др.); здесь только конфиг уровня: радиус карты, число жил, лимит волн, чекпоинты, шаги с триггерами и сообщениями.

## Ключи конфига уровня

| Ключ | Описание |
|------|----------|
| map_radius | Радиус гексагональной карты |
| ore_vein_count | Количество жил руды |
| wave_max | Лимит волн (0 = без лимита, основная игра) |
| is_tutorial | Режим обучения |
| tutorial_index | Индекс уровня обучения (0…4) |
| title | Название уровня |
| hints | Устаревшее; используется steps |
| checkpoint_count | Число чекпоинтов (-1 = 6, 0 = пустая карта без чекпоинтов) |
| steps | Массив подсказок: { "trigger": "id", "message": "текст" } |
| ore_consumption_multiplier | Множитель расхода руды (только для туториала, для гиперболизации) |

## Триггеры пошаговых подсказок

- **game_start** — старт игры
- **towers_5** — поставлено 5 башен
- **phase_selection** — фаза переключена на ВЫБОР
- **towers_saved_2** — сохранено 2 башни
- **phase_wave** — фаза переключена на ВОЛНУ
- **wave_started** — волна началась (враги пошли)
- **miner_on_ore** — майнер поставлен на руду
- **miners_3_on_ore** — три майнера на руде
- **miners_2_on_ore** — два майнера на руде (уровень 1: выдаётся Б А Б А А)
- **attacker_connected** — атакующая башня подключена к сети
- **ore_depletes** — руда истощилась
- **phase_build_after_wave** — снова фаза строительства после волны (уровень 0)
- **none** — не переключать шаг (последнее сообщение остаётся)

## Основная игра

**get_main_config()** — конфиг без туториала: map_radius из Config, 3 жилы, wave_max = 0, checkpoint_count = -1 (6 чекпоинтов), пустые steps.

## Уровни обучения (get_tutorial_levels)

| Индекс | Название | Описание |
|--------|----------|----------|
| 0 | Основы | Пустая маленькая карта (radius 8), без чекпоинтов. Цель: фазы, 5 башен → выбор 2 → волна; затем майнер на руду и атакующая рядом. 2 волны. |
| 1 | Энергия и руда | Карта как уровень 0, 3 жилы. Гиперболизированная трата руды (ore_consumption_multiplier: 5). Шаги: 2 майнера на руду, атакующие рядом → фаза волны → волна началась. 2 волны. |
| 2 | Стены и выбор | radius 11, 4 жилы, 6 чекпоинтов. 5 башен → выбор 2 → волна; стены блокируют путь. 3 волны. |
| 3 | Крафт | radius 11, 4 жилы. Подсказка про рецепты (B), три башни по рецепту → крафт (например TA+PA+NI=Silver). 3 волны. |
| 4 | Практика | Уменьшенная карта (radius 9), 3 жилы, 6 чекпоинтов. Первые 10 волн основной игры. Без пошаговых подсказок. |

**get_tutorial_level(index)** — возвращает конфиг уровня 0…4 или get_main_config() при неверном индексе.

## Где используется

Выбор конфига уровня и отображение подсказок по триггерам выполняет логика главного меню/загрузки уровня и UI туториала (проверка триггеров при действиях игрока и смена шага steps).
