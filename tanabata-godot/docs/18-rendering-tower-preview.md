# TowerPreview

## Назначение

Показывает полупрозрачный превью башни под курсором мыши. Позволяет видеть, какую башню поставишь и можно ли поставить.

## Когда показывается

Только в фазе BUILD. Только когда мышь над картой (гекс существует).

## Определение типа

В режиме отладки — из debug_tower_type. В обычном — по логике волн (майнер первая башня в блоках 1–4, иначе атакующая).

## Форма и цвет

- MINER — треугольник, жёлтый.
- ATTACK — круг, оранжевый.
- WALL — шестиугольник, серый.

Цвет меняется: зелёный оттенок если можно поставить, красноватый если нельзя.
