# InputSystem (система ввода)

## Назначение

Обрабатывает клики мыши и клавиши. Определяет, куда кликнули (гекс, UI), и ставит команды в очередь.

## Очередь команд

Чтобы не перегружать кадр, команды накапливаются и выполняются по несколько за кадр. Типы команд: place_tower, remove_tower, select_tower, toggle_selection.

## Клики мыши

**Левая кнопка:**
- BUILD: по пустому гексу — поставить башню; по башне — подсветить.
- SELECTION: по башне — переключить выбор (сохранить / не сохранять).

**Правая кнопка:**
- BUILD: по башне — удалить башню.
- WAVE / TOWER_SELECTION: по башне (не стена) — переключить вкл/выкл (**toggle_tower_enabled**). Выключенная башня не стреляет и не тратит руду, но остаётся в сети; майнер при выключении только передаёт сеть, не добывает.

## Определение гекса

Экранные координаты мыши переводятся в мировые через камеру и масштаб. Из мировых координат вычисляется гекс. Если гекса нет на карте — клик игнорируется.

## Размещение башни

Проверяется фаза, лимит башен, возможность поставить на гекс (проходимость, возможность размещения, отсутствие башни). Тип башни выбирается по логике волн или по режиму отладки. Создаётся сущность башни, обновляется тайл, башня добавляется в энергосеть.

## Удаление башни

Башня удаляется из ECS, тайл снова проходим, энергосеть пересобирается.

## Выбор типа башни

В обычном режиме: первые 4 волны в блоке 10 — первая башня майнер, остальные атакующие; следующие 6 волн — все атакующие. В режиме отладки (клавиши в game_root): **1** — случайная атакующая, **2** — майнер, **3** — стена, **6** — тестовая башня из списка (Config.DEBUG_TEST_TOWER_IDS: TOWER_QUARTZ, NI3, TA2, TOWER_LAKICHINZ); повторное нажатие 6 выключает режим. **0** — выключить любой дебаг-тип башни.

## Редактор энерголиний (режим U)

В игре доступен режим перетаскивания энерголиний: клавиша **U** включает/выключает **line_edit_mode** (в game_state). В этом режиме клики по карте обрабатывает **LineDragHandler**, а не размещение башен.

**Как пользоваться:**
1. Нажать U — включить режим редактирования линий (иконка «молния» в HUD).
2. Кликнуть по башне, у которой есть связи в энергосети: выбирается линия к соседней башне (по углу клика к центру башни). Линия временно скрывается (is_hidden).
3. Кликнуть по другой башне — линия перенаправляется (reconnect): старая связь удаляется, создаётся новая между «источником» перетаскивания и выбранной башней. Если reconnect невозможен (цикл, потеря питания), линия восстанавливается.
4. Клик по пустому гексу или по той же башне — отмена перетаскивания, линия снова видна.
5. Нажать U снова — выйти из режима, активное перетаскивание отменяется (cancel_line_drag).

LineDragHandler использует EnergyNetworkSystem.reconnect_line() и get_line_between_towers(); порог угла для попадания в линию — ANGLE_THRESHOLD (радианы).
