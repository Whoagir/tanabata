# OreGenerationSystem (генерация руды)

## Назначение

Создаёт залежи руды на карте процедурно. Руда — источник энергии для майнеров.

## Процесс генерации

1. Выбираются 3 центра залежей (с учётом зон исключения: entry, exit, checkpoints).
2. Общая мощность (240–270, из констант) распределяется между центрами (центральная жила, средняя, дальняя).
3. Центральная жила — центр + до 3 соседей (4 гекса). Остальные — гексы в радиусе 2 от центра.
4. Распределение: центральная — детерминированное по гексам; средняя и дальняя — «круги энергии» со случайными центрами и мощностью.
5. Для каждого гекса с ненулевой мощностью создаётся сущность ore: power, max_reserve, current_reserve, hex, radius, pulse_rate, is_highlighted.
6. Используется seed карты для воспроизводимости.

## Свойства руды

- **current_reserve** — текущий запас. Уменьшается когда майнеры питают башни.
- **max_reserve** — начальный запас.
- **power** — «сила» залежи (влияет на распределение).
- Ниже порога (0.1) руда считается истощённой, майнер на ней не активен.
