# NodePool (пул узлов)

## Назначение

Переиспользование узлов Godot вместо создания и удаления. Уменьшает нагрузку при большом числе сущностей (враги, снаряды).

## Как работает

- **acquire** — взять узел из пула. Если пул пуст — создать новый через фабричную функцию.
- **release** — вернуть узел в пул. Вызывается reset-функция (сброс позиции, масштаба).
- При инициализации можно предсоздать N узлов.

## Использование

EntityRenderer использует пулы для врагов и снарядов. При появлении сущности — acquire. При исчезновении — release. Узлы не удаляются, а переиспользуются.
