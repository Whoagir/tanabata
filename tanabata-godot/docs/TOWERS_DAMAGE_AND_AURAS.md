# Урон башен (ярус 0 и 1) и ауры скорости/урона

Источник: `data/towers.json`. Урон в игре: ярус 0 ×0.85, ярус 1 ×1.07 (скрипт `scripts/balance_tiers.py`).

| DE2  | 0           | 1.92                   | 1.92x                       |

## Ауры урона (DA1–DA6)

Дают плоский бонус к урону (+N к каждому выстрелу) союзным башням в радиусе. У самой башни урон 0.

| ID   | Урон (JSON) | Аура: damage_bonus | В игре (бонус к урону) |

| DA1  | 0           | 8                  | +8                     |

| DA3  | 0           | 29                 | +29                    |
| DA4  | 0           | 52                 | +52                    |
| DA5  | 0           | 86                 | +86                    |
| DA6  | 0           | 288                | +288                   |

## Прочие ауры (ярус 1–2)

| ID           | Эффект                          |
|--------------|----------------------------------|
| TOWER_QUARTZ | slow_factor 0.6, только по летающим (flying_only), radius 3 |
| TOWER_PARAIBA | all_enemies_slow: radius 4, slow 50%; при выходе из ауры — 30% slow на 2 с (exit_slow_factor 0.7, exit_slow_duration 2). Урон 0, атаки нет (NONE). Расход руды как DE4. |
| TOWER_DIPSEA | debuff_immunity, radius 3. Урон 0, атаки нет (NONE). |
